## 🚀 Getting Started
### Prerequisites
- Node.js 18+
### Installation
```bash
# Clone the repository
git clone https://github.com/Je0Dev/lang_website.git
cd lang_website
# Install dependencies
npm install
```
### Development
```bash
# Start development server
npm run dev
```
### Build
```bash
# Build for production
npm run build
```
